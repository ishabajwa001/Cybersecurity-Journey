# Ping Help (`-h`)

## Command

```bash
ping -h
```

## Description

Displays the help information for the `ping` command. It provides a summary of the command syntax and lists the available options supported by the installed version of `ping`.

## Purpose

* View the available command-line options.
* Learn the basic syntax of the `ping` command.
* Quickly access usage information without opening the manual page.

## Notes

* The output may vary depending on the operating system and the version of `ping`.
* On most Linux systems, `ping -h` and `ping --help` display the same help information.
* For detailed documentation, use:

```bash
man ping
```
